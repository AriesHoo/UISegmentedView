#UISegmentedView
--------------------------
####一、简介：

一个分段选择控件，支持样颜色及弧度设置

**1.1 Gradle集成**

```
allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```

```
dependencies {
     compile 'com.github.AriesHoo:UISegmentedtView:1.0.0'
}
```
